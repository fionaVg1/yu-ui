# @fiona_cai/yu-ui 组件库

### 快速开始
#### 1.安装组件库
```bash
npm i @fiona_cai/yu-ui
```
#### 2.引用组件库
```javascript
//全部引入
import '@fiona_cai/yu-ui/dist/css/index.css';
import YUI from '@fiona_cai/yu-ui';
Vue.use(YUI)

//按需引用
import '@fiona_cai/yu-ui/dist/css/demo.css';
import { Demo } from '@fiona_cai/yu-ui';
Vue.use(Demo);
```
